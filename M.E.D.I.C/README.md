# Medical-Chatbot 
# how to run?
Clone the repository 
```
git clone https://github.com/JyothikaPriyanka/Medical-Chatbot.git
```
### Step 1 : Create a conda environment after oping the repository 
```
conda create -n medibot python=3.10 -y
```
```
conda activate medibot
```
Step 2 : install the requirements 
```
pip install -r requirements.txt
```
